# test_nces
Тестовое задание для НЦЭУ - одностраничное приложение (SPA) для управления задачами.

## Описание проекта
Приложение для управления задачами с современным пользовательским интерфейсом и полным набором CRUD-операций. Разработано для эффективного управления статистическими задачами с расширенными возможностями фильтрации и сортировки.

## Функциональность
### Основные возможности:
- **Просмотр списка задач** - карточки с основной информацией
- **Создание и редактирование задач** - интуитивная форма с валидацией
- **Фильтрация задач** - по статусу, приоритету и меткам
- **Сортировка задач** - по дедлайну, приоритету и дате создания
- **Поиск задач** - по заголовку и описанию
- **Детальный просмотр задачи** - отдельная страница с полной информацией
- **Быстрое управление статусом** - изменение статуса прямо из списка
- **Удаление задач** - с подтверждением действия

### Структура задачи:
- Заголовок (обязательное поле)
- Описание
- Статус (`к выполнению` | `в процессе` | `выполненно`)
- Приоритет (`низкий` | `средний` | `высокий`)
- Дедлайн (дата)
- Метки (теги)
- Дата создания и обновления

## Технологический стек

- **React 18+** с функциональными компонентами и хуками
- **TypeScript** - строгая типизация
- **Material-UI (MUI)** - UI компоненты и иконки
- **Emotion & Styled Components** - стилизация
- **React Router v6** - навигация
- **Vite** - сборка и dev-сервер
- **Zustand** - управление состоянием
- **Axios** - HTTP-клиент
- **JSON Server** - моковый бэкенд
- **date-fns** - работа с датами
- **ESLint + Prettier** - линтинг и форматирование

### Предварительные требования:
- Node.js 20.0.0 или выше
- npm 8.0.0 или выше

### Установка и запуск:

1. **Клонирование репозитория:**
```bash
git clone https://github.com/ZlataGlushakova/test_nces.git
cd test_nces
```

2. **Установка зависимостей:**
```bash
npm install
```
3. **Запуск в режиме разработки:**
```bash
# Запуск приложения и мокового сервера одновременно
npm run dev:full
```
Или по отдельности:

```bash
# Терминал 1 - запуск мокового сервера
npm run server

# Терминал 2 - запуск приложения
npm run dev
```

### Приложение будет доступно по адресам:

Frontend: http://localhost:3000

JSON Server: http://localhost:3001

### Доступные скрипты

- npm run dev - запуск development сервера
- npm run dev:full - запуск приложения и сервера одновременно
- npm run server - запуск JSON Server с задержкой 500мс
- npm run build - сборка проекта для production
- npm run preview - предпросмотр собранного проекта
- npm run lint - проверка кода линтером
- npm run lint:fix - автоматическое исправление ошибок линтера
- npm run type-check - проверка TypeScript типов
- npm run format - форматирование кода Prettier
- npm run test - запуск тестов
